My main goals with this repository are to:

## How to Run the Files

Most files can be run directly with Python:

```bash
python3 filename.py
```

For example:

```bash
python3 chapter_02_variables_and_simple_data_types/name.py
```

Some later chapters may require additional libraries such as Pygame, Matplotlib, Plotly, Requests, or Django.
